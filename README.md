## Campaign thesis

**Pattern recognition without pattern imposition.**

The Director should be able to enter messy client operations, build a shared picture quickly, stabilize service and cash consequences, create stage-appropriate operating mechanisms, and transfer capability without creating permanent dependence on the consultant.

## Role-specific operating model

**Client Ops Runbook**

1. Read the System
2. Stabilize the Flow
3. Build the Mechanism
4. Transfer the Capability
