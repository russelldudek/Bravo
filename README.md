## Campaign thesis

**Operations leadership that builds capability.**

The campaign argues for pattern recognition without pattern imposition: read each growth-stage brand as it actually operates, stabilize the most consequential flow problems, build stage-appropriate mechanisms, and transfer capability to the client and Bravo team.

## Bravo-aware visual experience

The website uses employer-aware visual grammar derived from Bravo CPG’s official public presence:

- lowercase black `bravo` wordmark with a dark-green circular `CPG` device;
- black, white, and deep-green primary palette;
- clean contemporary sans-serif typography;
- practical category color accents;
- open, high-contrast editorial composition;
- a role-derived topographic Client Ops Runbook route rather than a generic dashboard.

The route animation and interactive Runbook tabs teach the operating model:

1. Read the System
2. Stabilize the Flow
3. Build the Mechanism
4. Transfer Capability

Keyboard navigation and `prefers-reduced-motion` behavior are included.

## Evidence and brand integrity

Brand provenance and asset decisions are documented in `BRAND-INTELLIGENCE.md`. The candidate site recreates the observed wordmark structure code-natively rather than copying or hotlinking a fragile logo asset.

## Research basis

Primary public sources include Bravo CPG’s official home, services, team, fractional COO, pricing, careers, and category pages, the canonical Airtable posting, and the user-supplied LinkedIn job-description text. Company-specific recommendations labeled as hypotheses are candidate points of view, not claims about Bravo CPG’s internal systems.

This is a candidate-created campaign and is not an official Bravo CPG publication or endorsement.
